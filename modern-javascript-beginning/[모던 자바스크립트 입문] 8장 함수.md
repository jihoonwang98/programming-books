## 8.1 함수 정의하기



### 8.1.1 함수를 정의하는 방법

- 함수를 정의하는 방법 4가지

  - **함수 선언문**

    `function square(x) { return x * x; }`

  - **함수 리터럴 (함수 표현식)**

    `var square = function(x) { return x * x; };`

  - **Function 생성자**

    `var square = new Function("x", "return x*x");`

  - **화살표 함수 표현식**

    `var square = x => x*x;`
